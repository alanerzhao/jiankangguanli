# Product Spec

## 产品定位

这是一个给个人长期使用的健康管理工具，重点是低摩擦记录和长期复盘，而不是医学诊疗。

## 核心用户

- 单个用户本人
- 每天记录饮水、睡眠、步数、热量、体重和心情
- 想在周、月、年尺度回看节奏变化

## 核心能力

1. 记录每日健康数据
2. 维护个人目标
3. 展示最近记录和趋势图
4. 展示 `7天 / 30天 / 本月 / 今年` 复盘
5. 导入导出 JSON 备份
6. 可选连接 Supabase 做自动云同步

## 关键用户路径

### 记录

用户打开页面，填写当天数据，保存后立即看到：

- 记录表更新
- 顶部摘要更新
- 趋势图更新
- 复盘卡片更新

### 复盘

用户切换复盘周期或趋势指标，快速判断：

- 这段时间记录是否稳定
- 目标达成率如何
- 睡眠、热量、体重是否在目标附近

### 云同步

用户提前在 `config.js` 配好 Supabase 后：

- 页面加载时尝试恢复会话
- 首次访问可自动触发 magic link
- 成功授权后自动拉取云端数据
- 本地新增记录或修改目标时自动写入云端

## 非目标

- 不做医疗建议系统
- 不做多人协作
- 不做复杂后台管理
- 不引入构建工具链和重型前端框架

## 成功标准

- 用户可以在 1 分钟内完成当天录入
- 用户可以在 10 秒内切换到任意周期复盘
- 在没有云端配置时，站点仍然可完整本地使用
